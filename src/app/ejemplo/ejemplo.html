Aplicación de ejemplo en Angular

Esta pequeña aplicación muestra una lista de conceptos básicos de Angular. Al hacer clic en cada concepto se despliega o se oculta su descripción.

  • {{ concepto.descripcion }}

Con este ejemplo vemos:

  • Cómo recorrer listas con *ngFor.
  • Cómo manejar eventos de clic con (click).
  • Cómo mostrar u ocultar contenido con *ngIf.

¿Cómo hemos construido esta aplicación de ejemplo?

Para crear esta pequeña aplicación se han seguido estos pasos:

  1. Crear un nuevo componente con Angular CLI: ng g c ejemplo.
  2. Definir en ejemplo.ts una lista de conceptos como un array de objetos de TypeScript, cada uno con nombre, descripción y un valor booleano para controlar si se muestra o no.
  3. En la plantilla ejemplo.html, recorrer la lista con la directiva *ngFor y usar (click) para manejar el evento de pulsar el botón.
  4. Utilizar la directiva *ngIf para mostrar u ocultar la descripción de cada concepto según el estado de la propiedad visible.

De esta forma se ve claramente cómo se conectan los datos definidos en TypeScript con la interfaz HTML a través del data binding de Angular.
