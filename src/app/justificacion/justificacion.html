¿Qué es Angular y por qué usarlo?

Angular es un framework de desarrollo creado por Google que se utiliza para construir aplicaciones web de una sola página (Single Page Applications o SPA) y aplicaciones web complejas que necesitan ser rápidas, organizadas y fáciles de mantener.

¿Para qué sirve?

Angular se utiliza principalmente para:

  • Crear interfaces de usuario dinámicas que se actualizan sin recargar la página.
  • Desarrollar aplicaciones web con muchas pantallas y componentes reutilizables.
  • Organizar mejor el código usando componentes, módulos y servicios.

¿En qué lenguajes se apoya?

Angular se basa en:

  • TypeScript: un superconjunto de JavaScript con tipos.
  • HTML: para definir la estructura de las vistas.
  • CSS: para el diseño y la apariencia de la aplicación.

¿Qué conocimientos previos son recomendables?

Antes de aprender Angular es recomendable tener:

  • Conocimientos básicos de HTML y CSS.
  • Nociones de JavaScript (variables, funciones, condicionales…).
  • Conocimientos muy básicos de la terminal y de uso de npm.

¿Por qué he elegido Angular para esta actividad?

Para esta actividad he elegido Angular porque:

  • Es un framework muy utilizado en el desarrollo profesional de aplicaciones web.
  • Ofrece una estructura muy clara del proyecto, lo que ayuda a aprender buenas prácticas.
  • Incluye herramientas integradas (ruteo, formularios, comunicación con APIs, etc.) que permiten escalar proyectos desde algo sencillo a algo mucho más complejo.

En resumen, Angular es una buena opción para aprender cómo se organiza una aplicación web moderna y qué herramientas ofrece un framework completo.
