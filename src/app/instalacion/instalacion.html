Instalación de Angular y creación del proyecto

Para poder trabajar con Angular necesitamos instalar algunas herramientas previas en nuestro ordenador. A continuación se describen los pasos básicos.

1. Instalar Node.js

Angular utiliza Node.js y npm (el gestor de paquetes de Node) para instalar dependencias y ejecutar comandos. El primer paso es descargar e instalar Node.js desde su página oficial. Al instalar Node también se instala npm.

2. Instalar Angular CLI

La Angular CLI (Command Line Interface) es una herramienta de línea de comandos que facilita la creación y gestión de proyectos Angular.

Una vez instalado Node, abrimos una terminal y ejecutamos: 

npm install -g @angular/cli

El parámetro -g indica que la instalación será global, es decir, que podremos usar el comando ng desde cualquier carpeta.

3. Crear un nuevo proyecto

En la carpeta donde queramos guardar el proyecto, ejecutamos: 

ng new tutorial-angular

Durante el proceso, la CLI puede hacer varias preguntas (por ejemplo si queremos añadir enrutador o algún tipo de estilo). Para esta actividad podemos aceptar las opciones por defecto o elegir:

  • Incluir enrutador: Sí, para poder tener varias páginas/secciones.
  • Estilos: por ejemplo, CSS.

Cuando el comando termina, se crea una carpeta llamada tutorial-angular con todo el esqueleto del proyecto.

4. Ejecutar la aplicación

Entramos en la carpeta del proyecto y ejecutamos el servidor de desarrollo: 

cd tutorial-angular
ng serve

Tras unos segundos, la aplicación estará disponible en el navegador en una dirección similar a: http://localhost:4200.

Cada vez que guardemos cambios en el código, la página se recargará automáticamente y veremos el resultado actualizado.

5. Visualizar los resultados en el navegador

Una vez creado el proyecto y dentro de la carpeta del mismo, se ejecuta el siguiente comando: 

ng serve --open

Este comando inicia un servidor de desarrollo y abre automáticamente el navegador en http://localhost:4200. Cada vez que se realizan cambios en los archivos del proyecto, Angular los detecta y recarga la página de forma automática.

6. Crear componentes adicionales

Para ampliar la aplicación y organizar el contenido en secciones, se pueden crear nuevos componentes con el siguiente comando de Angular CLI: 

ng generate component nombre-del-componente

En este tutorial se han creado varios componentes (Inicio, Justificación, Instalación, etc.) y se han enlazado mediante el sistema de rutas de Angular.
