Bienvenida al tutorial de Angular

Este sitio web es un pequeño tutorial interactivo sobre Angular, un framework de desarrollo para crear aplicaciones web modernas.

A lo largo de las distintas secciones aprenderás:

  • Qué es Angular y por qué se utiliza.
  • Qué necesitas instalar para empezar a trabajar con él.
  • Cómo crear tu primer proyecto y un componente sencillo.
  • Cómo funciona Angular “por dentro” a nivel básico.
  • Un ejemplo práctico de aplicación hecha con Angular.

El objetivo es que, al terminar el tutorial, tengas una idea general de cómo se estructura un proyecto en Angular y seas capaz de seguir por tu cuenta la documentación oficial.

Usa el menú de navegación superior para moverte entre las secciones del tutorial.
